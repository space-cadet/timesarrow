## Description
Compute Rényi entanglement entropies for subregions of the Z₂ gauge theory to verify topological order in the deconfined phase. This provides a smoking-gun signature: topological entanglement entropy γ = log(2) for deconfined, γ = 0 for confined.

## Acceptance Criteria

- [ ] Verify γ = log(2) in deconfined phase, γ = 0 in confined phase
- [ ] Compare with toric code theoretical prediction

## Implementation Details
The second Rényi entropy for a subregion A is:
```
S₂(A) = -ln Tr(ρ_A²)
```
where ρ_A is the reduced density matrix.

**Methods:**
1. **Swap operator (Monte Carlo):** S₂ = -ln⟨Swap_A⟩ where Swap_A exchanges two copies of the system in region A
2. **Replica trick:** Compute partition function Z_n for n copies with appropriate branch cuts

**Subregion shapes:**
- Cylinder: A = [0, L_x] × [0, L_y] × [0, ℓ] with ℓ < L_z
- Sphere: radius R < L/2

**Key diagnostic:**
For a topologically ordered phase:
```
S(A) = α|∂A| - γ + ...
```
where γ = log(D) and D is the total quantum dimension. For Z₂ gauge theory, D = 2, so γ = log(2).

**Expected results:**
- Confined phase: S(A) = α|∂A| (no topological term, γ = 0)
- Deconfined phase: S(A) = α|∂A| - log(2) (topological order, γ = log(2))
