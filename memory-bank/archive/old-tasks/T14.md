## Description
Build a minimal, static HTML web presentation for the Time's Arrow paper. Focus on speed of deployment and clarity over visual complexity. ## Implementation Details

### Tech Stack (KIRSS-Compliant)
- **Framework:** Static HTML5 (no build tools, no dependencies)
- **Styling:** Vanilla CSS with CSS variables for theming
- **Math:** KaTeX via CDN (async loading)
- **Components:** None — semantic HTML sections
- **Deployment:** Netlify (drop-in static site)

### Visual Layout Philosophy
- Single-page scroll experience (no routing complexity)
- Dark mode by default (quantum gravity aesthetic)
- Minimal animations (CSS transitions only, no JS frameworks)
- Content-first: 80% text/math, 20% visuals
- No interactive simulations (scope control)

### Content Structure
1. **Hero**: Paper title, author, abstract (simplified for lay readers)
2. **The Core Idea**: Why time has a direction — the Z₂ symmetry breaking concept in plain language
3. **The Mechanism**: How gauging time reversal works — math section with key equations
4. **Deep Dive**: Links to arXiv, supplementary calculations, and contact
