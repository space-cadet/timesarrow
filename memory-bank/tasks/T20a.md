# T20a: Phase 1 — 2D Square Lattice

*Created: 2026-06-24*
*Parent: T20*

## Objective
Validate the Monte Carlo algorithm on the simplest case where the exact solution is known.

## Parameters
- Lattice: 2D square, L = 8
- Thermal sweeps: 1000
- Measure sweeps: 5000
- Exact critical coupling: β_c = ½ ln(1+√2) ≈ 0.4407

## Observables
- ⟨P⟩(β) — average plaquette
- Error bars from binning + jackknife
