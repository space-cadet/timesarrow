## Description
Couple the Z₂ gauge field to a scalar (Higgs-like) field living on vertices and study whether matter coupling can spontaneously break the remaining Z₂ symmetry, selecting a global arrow of time. This addresses the gap between "time-orientability" (what the paper currently explains) and a true "arrow of time" (which requires a preferred direction).

## Implementation Details
**Scalar field action:**
```
S_ϕ = Σ_v [½(∇_μ ϕ_v)² + V(ϕ_v) + λ ϕ_v τ_v]
```
where τ_v is the local time-orientation variable and λ is the coupling strength.

**Potential:** V(ϕ) = ½m²ϕ² + ¼gϕ⁴ (symmetric double-well for m² < 0)

**Key observables:**
1. ⟨ϕ⟩: Non-zero indicates spontaneous symmetry breaking
2. ⟨τ_v⟩: Should remain zero (Elitzur's theorem) but correlators show ordering
3. Dressed correlator C(r) with matter: ⟨ϕ_v τ_v ∏σ_e τ_w ϕ_w⟩

**Phase diagram:**
- Small λ: Matter decouples, recover pure Z₂ LGT phases
- Large λ: Matter forces orientation selection
- Critical λ_c(K): Phase boundary

**Physical interpretation:**
If ⟨ϕ⟩ ≠ 0 and correlates with τ_v, this demonstrates that matter can "choose" a global arrow of time from the orientable deconfined phase. This completes the story from orientability to arrow.

## Notes

- If this simulation succeeds, it provides a compelling narrative: (1) Z₂ LGT gives orientability, (2) matter coupling selects the arrow.
