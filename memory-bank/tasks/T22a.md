## What This Task Actually Computes

This task computes a **normalized SU(2) four-leg group average**:

$$G(j) = \\frac{1}{(2j+1)^3} \\int_{SU(2)} dh \\,[\\chi^j(h)]^4 = \\frac{1}{(2j+1)^2}$$

### Analytic Derivation (Primary Result)

Using the Clebsch–Gordan series:
$$\\chi^j(h)^2 = \\sum_{J=0}^{2j} \\chi^J(h)$$

Therefore:
$$I(j) = \\int_{SU(2)} dh \\,[\\chi^j(h)]^4 = \\sum_{J=0}^{2j} \\sum_{K=0}^{2j} \\int_{SU(2)} dh \\,\\chi^J(h)\\chi^K(h) = \\sum_{J=0}^{2j} 1 = 2j+1$$

by orthogonality of characters. With the $(2j+1)^{-3}$ normalization factor:
$$G(j) = \\frac{2j+1}{(2j+1)^3} = \\frac{1}{(2j+1)^2}$$

### Key Results

- $G(j=\\tfrac{1}{2}) = \\frac{1}{4} = 0.25$
- $G(j=1) = \\frac{1}{9} \\approx 0.111$
- Exact ratio: $G(1)/G(\\tfrac{1}{2}) = \\frac{4}{9} \\approx 0.444$
- Asymptotic: $G(j) \\sim \\frac{1}{4j^2}$ for large $j$

Monte Carlo (2M samples) validates these to $<0.1\%$.

## What This Is NOT

- **NOT** a complete FK or EPRL spin-foam vertex amplitude.
- **NOT** evidence for $j=1/2$ dominance in any physical model.
- **NOT** a substitute for the full intertwiner, Immirzi parameter, and face-amplitude structure of a real spin-foam model.

The original code mislabeled this as an "FK vertex amplitude" and applied an unsupported extra squaring step ($0.45 \\to 0.20$). Both errors have been corrected.
