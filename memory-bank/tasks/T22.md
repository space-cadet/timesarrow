## Overview

Compute spin foam vertex amplitudes to validate the j=1/2 dominance assumption in the timesarrow manuscript. This addresses the deepest T18 gap: deriving (not assuming) the effective Wilson action from LQG first principles.

## Physics Goal

Compute |A_v(j₁, j₂, j₃, j₄)|² for:
1. Uniform j=1/2: (1/2, 1/2, 1/2, 1/2)
2. Uniform j=1: (1, 1, 1, 1)
3. Mixed: (1/2, 1/2, 1/2, 1)
4. Higher j: (3/2, 3/2, 3/2, 3/2)

If |A_v(j=1)|² / |A_v(j=1/2)|² ≪ 1, then j=1/2 dominates, validating the manuscript's key assumption.

## Why This Isn't "Obvious"

See [implementation-details/spin-foam-amplitude-calculation.md](../implementation-details/spin-foam-amplitude-calculation.md) for detailed reasoning. Key points:

1. **FK vertex structure:** The amplitude involves a nontrivial kernel K_j(g₁,g₂,g₃,g₄) with γ-dependence
2. **Intertwiner multiplicity:** Number of intertwiners grows with j; coherent state integrals have nontrivial width scaling
3. **EPRL vs FK distinction:** Different asymptotics matter for quantitative claims
4. **Compounding:** Single-vertex suppression is a conservative lower bound for the full path integral
