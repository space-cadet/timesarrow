## Objective

Analyze flux loop statistics from saved Z₂ configurations to give a geometric characterization of the deconfinement transition.

## What Is Measured

From saved link configurations (T34a output):
1. Compute plaquette flux B_p = ∏_{e∈∂p} σ_e
2. Identify connected clusters of B_p = −1 plaquettes
3. Measure statistics

## Primary Observables (Unambiguous)

| Observable | Definition | Phase Behavior |
|-----------|-----------|---------------|
| Connected component size distribution | P(S) vs S | Confined: large components; deconfined: small |
| Winding probability | P_wind around each H₁ generator | Confined: finite; deconfined: → 0 |
| Maximum component size | S_max / N_p | Confined: O(1); deconfined: → 0 |

## Secondary Observables (Convention-Dependent)

- **Loop-length distribution:** Requires a pairing convention at degree-≥4 dual vertices where multiple loops meet. Report with clear label: "loop-length distribution (pairing-convention dependent)."

## Phase Identification (Corrected)

| Phase | Flux Behavior |
|-------|--------------|
| **Low β (confined)** | Flux defects dense, large connected components, system-spanning/winding loops probable |
| **High β (deconfined)** | Flux loops suppressed, small components, no winding |

## Framing

> "Geometrical characterization of flux proliferation" — **not** an order parameter, **not** a CZX test.

## Notes

**Key correction from review:** The original plan had the phase identification backwards (confined = small loops, deconfined = winding). Corrected: in the confined phase, flux defects are dense and can form large winding loops; in the deconfined phase, flux loops are suppressed and small.

**Another correction:** "Connected component size" and "winding" are unambiguous. A unique decomposition into individual loops is not — dual vertices can have degree ≥ 4, so multiple loops meet. Report component/winding statistics as primary; label any loop-length distribution as convention-dependent.
