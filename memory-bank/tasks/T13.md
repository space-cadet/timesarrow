## Description
Develop a high-fidelity, interactive, and accessible single-page web presentation for the "Time's Arrow" research paper. The goal is to translate complex concepts (LQG, SPT phases, $Z_2$ gauging) into a visual narrative for non-experts while maintaining academic rigor for deep-divers.

## Acceptance Criteria
- [ ] Initialize Next.js 14+ scaffold in `/web-presentation` with Tailwind CSS and MDX.
- [ ] Implement a "Quantum Gravity" theme (Dark mode, deep space palette).
- [ ] Create a "Hero" section with an interactive "Arrow of Time" toggle.
- [ ] Draft and implement the three-tier storytelling structure (Hook → Core → Technical).
- [ ] Integrate KaTeX for high-quality mathematical rendering.
- [ ] Export and embed at least 3 key TikZ diagrams as interactive SVGs/React components.
- [ ] Deploy the initial version to Netlify/Vercel.

## Implementation Details
- **Tech Stack**: Next.js (App Router), Tailwind CSS, Framer Motion (animations), MDX, KaTeX.
- **Storytelling Tiers**:
  1. **Hook**: The mystery of time's direction.
  2. **Core**: Visualizing $Z_2$ symmetry breaking on spin networks.
  3. **Technical**: Mapping microscopic SPT phases to macroscopic tetrads.
- **Interactive Component**: A global state toggle for "Symmetry" vs "Broken" that updates visualizations across the page.
