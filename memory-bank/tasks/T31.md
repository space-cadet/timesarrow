After implementing `gauge_invariant_signed_volume_3d()` and the `--gauge-invariant-signed-volume` CLI flag, and running test simulations on L=6 and L=8:

| Test | Result |
|------|--------|
| Cold start (all links +1) | Q_GI = 1 ✓ |
| L=6 thermalized | Q_GI ≈ 0.02–0.08 (weak trend, wrong magnitude) |
| L=8 thermalized | Q_GI ≈ 0.01 (flat, no phase discrimination) |

**Root cause:** The path products `P(0→r)` involve O(L) links. By Elitzur's theorem, individual link variables are random ±1 in any thermalized state, so path products are ±1 with equal probability. The triple product `s(r₁)·W·s(r₂)` then averages to ~0 regardless of phase. This is a **fundamental obstruction**, not a statistics or implementation issue.

**Decision:** Abandon the signed volume as a deconfinement order parameter. Use the **Polyakov loop** (already implemented) as the standard gauge-invariant order parameter. The Polyakov loop measures the free energy of a static charge pair and is the canonical diagnostic for deconfinement in Z₂ gauge theory.

**2026-07-16:** The project scope was clarified. The main open issue is the quantum-geometric interpretation: the intertwiner/$\hat Q$/CZX construction remains analytic and interpretive, while the direct gauge-invariant dressed orientation correlator is a documented negative result rather than a successful observable.

## Negative Result (Worth Recording)

**Claim:** The signed volume operator $\hat{Q}$ cannot serve as a deconfinement order parameter in Z₂ lattice gauge theory because it is either gauge-dependent (original construction) or averages to zero in any thermalized state (gauge-invariant dressed correlator construction).

**Evidence:**
1. The gauge-dependent `signed_volume_3d()` gives |Q|/N ~ 0.03–0.09, but the value depends on the arbitrary gauge choice.
2. The gauge-invariant `gauge_invariant_signed_volume_3d()` (dressed correlator: Q_GI = (1/N²) Σ s(r₁)·W(r₁→r₂)·s(r₂)) gives Q_GI ≈ 0.01 for all β values on thermalized L=8 configurations, with no phase discrimination.
3. The cold-start test (all links +1) gives Q_GI = 1, confirming the implementation is correct — the failure is physical, not a bug.
4. Archived hot-start/cold-start calibration files from 2026-07-14 show the same obstruction more explicitly: plaquette and Wilson-loop observables order strongly by `β=1.0`, while `Q_GI` can still collapse near zero and vary strongly across seeds/sectors.

**Implication:** The connection between "arrow of time" and "signed volume" in the manuscript must be reframed. The arrow of time cannot be diagnosed by a local operator like $\hat{Q}$ in a gauge theory. A gauge-invariant diagnostic like the Polyakov loop (or Wilson loop area law) is required.

## Polyakov Loop Results (2026-07-14)

After pivoting from the signed volume to the Polyakov loop, proof-of-principle scans were run for L=8, 10, 12 across the transition region (β = 0.60–0.85, 10k thermal + 10k measure sweeps).

### Key Finding: Polyakov Loop Susceptibility Peak

| L | β_peak | χ_P (peak) | Growth |
|---|--------|-----------|--------|
| 8 | 0.76 | 355 | — |
| 10 | 0.76 | 621 | 1.75× |
| 12 | 0.76 | 886 | 2.50× |

The susceptibility peak is consistently at **β = 0.76**, matching the known critical point for 3D Z₂ gauge theory. The peak height grows with system size, consistent with a second-order transition.

### Mean Polyakov Loop ⟨P̄⟩

In the disordered phase (β < 0.70), ⟨P̄⟩ ≈ 0.
In the ordered phase (β > 0.80), |⟨P̄⟩| → 1.
The sign is random due to Z₂ symmetry (system spontaneously chooses + or −).

### Binder Cumulant U_P

At β = 0.76 (peak):
- L=8: U_P = 0.634
- L=10: U_P = 0.640
- L=12: U_P = 0.614

In the deep ordered phase (β = 0.85), U_P → 0.665 (close to the 3D Ising universal value of 2/3 ≈ 0.667).
