## Objective
Extract critical exponents and characterize the phase transition order in 3D Z₂ LGT.

## Key Finding
**3D Z₂ LGT has a FIRST-ORDER phase transition**, not a continuous one:

| Evidence | Result |
|----------|--------|
| Binder cumulant minimum scaling | U_min = 2/3 - 2.81·L^(-3.198), exponent ≈ -3 |
| Peak heights | χ_max → ~0.70, C_V,max → ~0.52 (constants, not diverging) |
| Peak widths | FWHM limited by grid resolution (Δβ = 0.02) |
| Scaling collapse | Fails with 3D Ising exponents (curves don't collapse) |
