## Objective

Audit whether an explicit CZX-like microscopic construction can be realized on a 4-valent 3D complex. This is the genuinely quantum step that validates (or refutes) the CZX correspondence.

## Why This Is Hard

The original CZX model (Chen, Liu, Wen) is a **2D global-Z₂ SPT**, not a toric-code model. Its relationship to a 3D gauged model must be constructed and tested, not inferred from ordinary toric-code stabilizers. The naive proposal:

$$H = -\sum_v A_v - \sum_p B_p - J \sum_e \sigma^z_e$$

is just a perturbed toric code. The projector $\prod_v (1+A_v)/2$ projects to the gauge-invariant sector, **not** the CZX subspace. And a deconfined gauge state does not automatically have a nontrivial ungauged SPT invariant.

## Corrected Audit Steps

### Step 1: Specify Local Hilbert Space

- **Four qubits per 4-valent vertex** (not edge qubits; not mixed)
- Each vertex has a local Hilbert space $\mathcal{H}_v = (\mathbb{C}^2)^{\otimes 4}$

### Step 2: Write the CZX-Inspired Local Symmetry

- Explicitly write the global symmetry operator with controlled-Z structure
- For the original 2D CZX model: global Z₂ acts as $\bigotimes_v U_v$ where $U_v$ involves controlled-Z between neighboring vertex qubits
- Extend to 3D: same local action on a 4-valent 3D graph
- **Verify:** Does this symmetry preserve the proposed physical/projected subspace?

### Step 3: Define the Candidate State

- Construct $|\Psi\rangle$ on the chosen 4-valent complex using the CZX construction

### Step 4: Apply Intertwiner Projection

- Apply the intended SU(2)/intertwiner projection, if any
- **Test:** Is the projected state nonzero and symmetry-preserving?

### Step 5: Derive Parent Hamiltonian

- Derive candidate local parent terms **from the state**, not by postulating stabilizers + perturbations
- Does a local parent Hamiltonian exist for this state?

### Step 6: Select Diagnostic

- Only then decide whether an SPT diagnostic exists, or whether the result is a gauged theory with Wilson/'t Hooft operators

## Acceptance Criteria

| Outcome | Interpretation |
|---------|---------------|
| State is nonzero, symmetry-preserving, has local parent H, SPT diagnostic exists | **Success:** Explicit 3D CZX realization |
| State is nonzero but no local parent H exists, or no SPT diagnostic | **Partial:** Structural correspondence remains; microscopic realization is obstructed |
| State is zero or not symmetry-preserving | **Negative result:** The claimed correspondence cannot be realized microscopically |

All three outcomes are useful and publishable.

## Notes

**Key correction from review:** The original plan proposed the incorrect projector $P_{\text{CZX}} = \prod_p (1+B_p)/2 \prod_v (1+A_v)/2$. This projects to the gauge-invariant toric-code subspace, not the CZX subspace. The correct order is: specify symmetry → define state → test projection → derive Hamiltonian → select diagnostic.

**Scope check:** Exact diagonalization on a 3D diamond lattice is likely infeasible (L=4 has too many states). Options:
- 2D layer (effective square lattice with 4-valent decoration) where ED is feasible
- Small tensor network (PEPS) with small bond dimension
- Analytic audit without numerics
