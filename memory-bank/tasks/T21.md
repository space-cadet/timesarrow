## Description
Construct and verify the ground state of the CZX-like spin-network state using Projected Entangled Pair States (PEPS) in 3D. The GPT 5.5 review flagged the CZX/SPT correspondence as "suggestive but overstated" — this simulation provides concrete ground-state evidence that the spin-network state has the correct SPT structure.

## Implementation Details
The CZX plaquette state on the spin network is:
```
|ψ_p⟩ = |0000⟩_p + |1111⟩_p
```
where the four qubits belong to four neighboring vertices (Eq. 64-65 in timesarrow.tex).

**PEPS construction:**
1. Build 6-index local tensor A^{i}_{u,d,l,r,p} where i is physical index (intertwiner qubit), u,d,l,r are virtual bond indices, and p is the plaquette projector index
2. Virtual bond dimension D = 2 (matching j=1/2)
3. Impose plaquette constraints via auxiliary tensors

**Key diagnostics:**
- **Gap:** Δ = E₁ - E₀ > 0 (gapped ground state)
- **Entanglement entropy:** S₂(A) = α|∂A| - γ + ... with γ = log(2) for topological order
- **String order parameter:** Non-zero for SPT phase, zero for trivial phase
- **Transfer matrix spectrum:** Check for topological degeneracy on torus

**Tools:** ITensor, TeNPy, or custom PEPS code. 3D PEPS is computationally expensive; may need to use boundary MPS methods or variational optimization.

**Expected results:**
- Gapped spectrum confirming SPT stability
- γ = log(2) confirming deconfined Z₂ topological order
- Non-zero string order parameter distinguishing non-trivial SPT from trivial phase
