## Gate 1 Results: Four-Valent Square-Lattice Test

### L=2 (Dense Exact Diagonalization) — ✅ PASS

- **Hilbert space:** 8 edge-qubits, dim = 256
- **Ground state energy:** E₀ ≈ 1.2×10⁻¹² ≈ 0
- **Intertwiner subspace dimension:** 1
- **<ψ|U_X|ψ>:** +1.000000
- **<ψ|U_CZ|ψ>:** +1.000000
- **First excited state:** E₁ ≈ 2.20, gap ΔE ≈ 2.20

### L=3 (Power Iteration) — ❌ FAIL

- **Hilbert space:** 18 edge-qubits, dim = 262,144
- **Ground state energy:** E₀ ≈ 3.684 (converged, stable)
- **No exact intertwiner subspace exists**

### Critical Finding

The **edge-qubit model is incompatible** with simultaneous singlet constraints at all vertices for L≥3. The intersection of all vertex singlet projectors is EMPTY for L=3 (best approximation has ~3.68 unsatisfied vertices on average).

This is a **negative result** for the edge-qubit Levin-Wen type construction.

## Pivot: Vertex-Qubit Model

**Realization:** T35a uses **vertex-qubits** (qubits at plaquette corners), not edge-qubits. The diamond-lattice CZX construction likely requires:
- **Option A:** Vertex-qubits (qubits at diamond-lattice vertices / hexagon corners)
- **Option B:** Vertex-hexagon incidence qubits (qubits associated with vertex-hexagon pairs)
- **Option C:** Some other placement not yet considered

The edge-qubit model tested above is a Levin-Wen/string-net type construction, which is different from the T35a CZX construction.

## Hamiltonian Bug Note

Initial implementations had a bug: computed `H = I - ΣPᵥ` instead of `H = Nᵥ·I - ΣPᵥ`. The correct Hamiltonian is:

```
H = Σᵥ(I - Pᵥ) = Nᵥ·I - Σᵥ Pᵥ
```

This was fixed in all Rust implementations.

## Context

The connected diamond 2-skeleton and hexagon enumeration from T33a are sufficient for the first state-construction tests; its missing 3-cells prevent closed-3-manifold and bulk-topology claims. T35a established that the literal four-qubit CZX operator leaks from the intertwiner subspace, so full-Hilbert-space invariance is not sufficient. The two-valent square-loop result is a toy-model result only and must not be promoted to the four-valent or diamond setting.

The task has three acceptable outcomes: a compatible candidate, a partial construction with a precisely stated missing gate, or a reproducible obstruction. Neither a toric-code projector nor ordinary gauge-transition numerics may substitute for a microscopic CZX realization.
