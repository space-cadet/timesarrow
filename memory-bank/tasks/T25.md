## Implementation

### ts-quantum Library Changes
- **index.ts**: Exported intertwiner module
- **volumeOperator.ts**: Fixed `checkZ2Structure()` to handle zero eigenvalues
  - Old: Paired by sorted absolute value (broke on zeros)
  - New: Groups by absolute value, checks equal numbers of +q and -q
- **nValent.ts**: Added parity check in `constructNValentBasis()`
  - Odd number of half-integer spins → returns dimension 0 (can't couple to J=0)

### Geometric Embeddings
- `createGeometricEmbedding()` — general embedding interface
- `buildGeneric6ValentEmbedding()` — triangular prism (6 non-coplanar edges)
