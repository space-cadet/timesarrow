## Objective
Simulate Z₂ gauge theory on 3D cubic lattice — the paper target for confinement-deconfinement transition.

## Effective Action
```
S_Z₂[σ] = -K Σ_p ∏_{e∈∂p} σ_e
```
where σ_e ∈ {+1, -1} lives on edges of a cubic lattice, and the sum runs over plaquettes.

## Key Observables
1. Wilson loop: W(γ) = ⟨∏_{e∈γ} σ_e⟩
2. Dressed correlator: C(r) = ⟨τ_v ∏_{e∈γ_{vw}} σ_e τ_w⟩ (gauge-invariant relative orientation)
3. Specific heat: C_v = (∂²lnZ/∂K²) / L³
4. Binder cumulant for precise K_c determination

## Algorithm
Standard Metropolis with local updates on edge variables.
- Thermalization: 10⁴ sweeps
- Measurements: 10⁵ sweeps with decorrelation analysis
- Lattice sizes: L = 8, 12, 16, 20 (finite-size scaling)

## Wilson Loop Results
- Area law vs perimeter law transition at β_c
- String tension σ(L) from W(γ) ~ exp(-σ·Area)
